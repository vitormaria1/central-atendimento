# Modelos de Documentos — Placeholders

Esses arquivos `.docx` foram sanitizados para uso como modelos.

## Placeholders

- `{{CPF}}`
- `{{CNPJ}}`
- `{{CEP}}`
- `{{TELEFONE}}`

## Observações

- Os placeholders são substituições seguras (evitam deixar CPF/CNPJ/CEP/telefone de clientes).
- Se você quiser ampliar os placeholders (ex.: `{{NOME_CLIENTE}}`, `{{ENDERECO}}`, `{{EMAIL}}`), eu posso evoluir o sanitizador.
